# Collect It

Collect It is a free Pokémon TCG analytics MVP built with Next.js, Firebase, Gemini, and PokeWallet.

## Included features
- Firebase Auth with Google and email/password sign-in
- AI card scan endpoint using Gemini 2.5 Flash
- PokeWallet-powered market lookup with fallbacks
- Firestore binder storage with per-user isolation
- Portfolio dashboard with eBay deal links

## Setup
1. Install dependencies with `npm install`.
2. Copy `.env.example` to `.env.local` and fill in your API keys.
3. Run `npm run dev`.
